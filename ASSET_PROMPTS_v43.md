# v43 追加素材プロンプト

このバージョンでは、主人公アイコンの表情差分を増やせるようにしています。  
ファイルが無い場合は既存画像へ自動でフォールバックします。

## 1) 主人公の表情差分（推奨）

保存先: `assets/img/`  
形式: **PNG / 背景透過**  
推奨サイズ: **1024 x 1024 px**  
構図: **顔が大きめ、円形トリミングしても見切れにくい、胸上まででも可**

### 共通ビジュアル指定
- 若い女性配信者 / Vtuber風
- スマホ縦画面のゲームHUDで小さく表示しても顔が読める
- 輪郭くっきり、目と口の差が分かりやすい
- 背景は完全透過
- 髪飾りや配信者らしいアクセントがあると良い
- ネオン配信空間に合う、ピンク・シアン系の映え
- 塗りは厚塗りすぎず、UIに馴染む高コントラスト

### `char_focus.png`
- 用途: 通常より集中している時
- 表情: 真剣、目線が前、少し口を結ぶ、勝ち筋を読んでいる感じ
- 生成プロンプト:
  - **日本の若い女性配信者キャラクター、Vtuber風、胸上、正面寄り、集中している表情、目はやや鋭く、口は小さく結ぶ、ネオン配信映え、ピンクとシアンの差し色、背景透過、ゲームUIアイコン向け、高コントラスト、円形トリミング前提、1024x1024**

### `char_warning.png`
- 用途: トレンド変更直前
- 表情: 少し驚き、気づいた顔、目を開く、口は小さく開く
- 生成プロンプト:
  - **日本の若い女性配信者キャラクター、Vtuber風、胸上、正面寄り、トレンドの変化に気づいた表情、少し驚き、目を大きく開く、口は小さく開く、かわいさと緊張感の中間、ネオン配信映え、背景透過、ゲームHUD用、円形トリミング前提、1024x1024**

### `char_hype.png`
- 用途: 連鎖、急上昇、バーストなど見せ場
- 表情: テンション高い笑顔、口大きめ、目も輝く
- 生成プロンプト:
  - **日本の若い女性配信者キャラクター、Vtuber風、胸上、正面寄り、配信の山場で盛り上がっている表情、明るい笑顔、目が輝く、口を大きめに開ける、ライブ感、熱量、ピンクとシアンのネオン反射、背景透過、ゲームUIアイコン向け、円形トリミング前提、1024x1024**

### `char_panic.png`
- 用途: 満杯寸前、炎上中、ゲームオーバー直前
- 表情: 焦り、冷や汗、でも見苦しすぎない
- 生成プロンプト:
  - **日本の若い女性配信者キャラクター、Vtuber風、胸上、正面寄り、かなり焦っている表情、目を見開く、冷や汗、小さく口を開ける、パニック寸前だがかわいさは残す、配信事故寸前の空気、背景透過、ゲームHUDアイコン向け、高コントラスト、円形トリミング前提、1024x1024**

## 2) 任意のSE素材（あるとさらに派手）

保存先: `assets/audio/`  
形式: **WAV / 48kHz / 24bit / stereo でも mono でも可**

### `se_trend_alert.wav`（任意）
- 用途: 切り替わる前の警告
- 長さ: **0.4〜0.8秒**
- プロンプト:
  - **未来的なゲームUI警告SE、短い上昇音、シアンのネオン感、配信向け、聞き取りやすい、やりすぎない、0.6秒前後**

### `se_trend_shift.wav`（任意）
- 用途: トレンド切替時の派手な演出
- 長さ: **0.8〜1.3秒**
- プロンプト:
  - **配信映えするトレンド切替SE、ネオンのフラッシュ感、短いインパクト、上昇→着地の気持ちいい決め音、ピンクとゴールドの印象、1.0秒前後**
